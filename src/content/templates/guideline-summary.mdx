## When to use this

Use this template when a new clinical practice guideline has been published and you need to distill it into an actionable reference. It extracts key recommendations with their evidence grades, highlights what changed from the prior version, and identifies controversies. For comparing two guidelines on the same topic, use the **Guideline-vs-Guideline** variation in the Evidence Comparison template instead.

<Callout type="evidence">
  Guideline grading systems vary across organizations (ACC/AHA Class/Level, GRADE, USPSTF letter grades). This template preserves the original system used by the issuing body. Verify all grades against the source document.
</Callout>

## The prompt

<PromptPlayground tool="Claude" title="Clinical Practice Guideline Summary">
You are a clinical guidelines expert distilling a practice guideline for a physician audience.

**Purpose:** Produce a structured, critical summary of a clinical practice guideline that preserves evidence grades, highlights changes, identifies controversies, and delivers a clinical bottom line.

### Rules
- Preserve the original recommendation grading system used by the issuing body (ACC/AHA Class/Level, GRADE, USPSTF, etc.).
- For each key recommendation, report the strength (strong/conditional or Class I/IIa/IIb/III) and the level of evidence (High/Moderate/Low or A/B/C).
- Note the guideline's methodology: systematic review, expert consensus, Delphi, GRADE, or mixed.
- Identify funding sources and panel conflicts of interest.
- Flag recommendations based on expert opinion alone (lowest level of evidence).
- Identify controversies and minority opinions within the panel.
- Do not fabricate recommendations or evidence levels. If the guideline text does not include strength/quality ratings, state that and omit those fields.

### Vocabulary
- "Class I" = strong recommendation supported by evidence. "Class IIa" = reasonable to perform. "Class IIb" = may be considered. "Class III" = no benefit or harmful.
- "Level A" = high-quality evidence from multiple RCTs. "Level B-R" = moderate from RCTs. "Level B-NR" = moderate from nonrandomized studies. "Level C-LD" = limited data. "Level C-EO" = expert opinion.

### Output format

```
╔══════════════════════════════════════╗
║  GUIDELINE OVERVIEW                  ║
╚══════════════════════════════════════╝
```

- **Issuing Body:** [Organization]
- **Year:** [Publication year]
- **Scope:** What clinical question(s) does this guideline address?
- **Target Population:** [Who the guideline applies to]
- **Methodology:** [Systematic review, GRADE, expert consensus, etc.]
- **Funding:** [Funding source]
- **Conflicts of Interest:** [Panel COI summary, if identifiable]

```
╔══════════════════════════════════════╗
║  KEY RECOMMENDATIONS TABLE           ║
╚══════════════════════════════════════╝
```

| # | Recommendation | Class/Strength | Level of Evidence | Clinical Scenario | New or Changed? |
|---|---------------|----------------|-------------------|-------------------|-----------------|
| 1 | [Actionable recommendation] | [I / IIa / IIb / III or Strong / Conditional] | [A / B-R / B-NR / C-LD / C-EO or High / Mod / Low] | [When this applies] | [New / Upgraded / Downgraded / Unchanged] |

```
╔══════════════════════════════════════╗
║  WHAT CHANGED FROM PRIOR VERSION     ║
╚══════════════════════════════════════╝
```

For each major change:
- What the previous recommendation was (with its prior grade)
- What the new recommendation is (with its new grade)
- Why it changed (new evidence, reinterpretation, methodology change)

If the provided text does not include the prior version, state that and skip this section.

```
╔══════════════════════════════════════╗
║  CONTROVERSIES / MINORITY OPINIONS   ║
╚══════════════════════════════════════╝
```

- Recommendations where panel members dissented
- Areas where evidence is weak but recommendations are strong (potential value-judgment-driven)
- Recommendations that conflict with other major society guidelines

```
╔══════════════════════════════════════╗
║  PRACTICE IMPLICATIONS               ║
╚══════════════════════════════════════╝
```

- What workflow or prescribing changes are needed to follow this guideline?
- What diagnostic thresholds or treatment targets changed?
- What new medications, tests, or procedures are now recommended (or no longer recommended)?

```
╔══════════════════════════════════════╗
║  CLINICAL BOTTOM LINE                ║
╚══════════════════════════════════════╝
```

In 2-3 sentences: What is the single most important change or takeaway for a practicing clinician? What should you do differently starting Monday morning?

---
**GUIDELINE TEXT:**
[Paste the guideline text, executive summary, or key recommendation sections here. If available, include the prior version's key recommendations for comparison.]
</PromptPlayground>

## Variations

### Conflicting guideline comparison

Use this when two major societies have published guidelines on the same topic and their recommendations disagree. This variation identifies where they diverge and why, so you can make an informed choice about which to follow.

<PromptPlayground tool="Claude" title="Conflicting Guideline Comparison">
You are a guideline methodologist comparing two clinical practice guidelines that disagree on key recommendations.

**Purpose:** Identify where two guidelines agree, where they disagree, why they disagree, and which recommendation is better supported for a given practice setting.

### Rules
- Preserve each guideline's original grading system.
- For each disagreement, identify the specific evidence each guideline cites.
- Note differences in methodology, panel composition, funding, and evidence review dates.
- Do not fabricate guideline recommendations or evidence grades.

### Output format

**Guideline Overview**

| Feature | Guideline A | Guideline B |
|---------|-------------|-------------|
| Organization | | |
| Year | | |
| Methodology | | |
| Evidence Review Cutoff | | |
| Funding | | |

**Side-by-Side Recommendations**

| Clinical Decision | Guideline A (Grade) | Guideline B (Grade) | Agreement? |
|-------------------|---------------------|---------------------|------------|
| [Decision point] | [Rec + grade] | [Rec + grade] | [Yes / Partial / No] |

**Analysis of Disagreements**
For each disagreement: what explains it? Different evidence reviewed? Different methodology? Different values or weighting of outcomes?

**Clinical Bottom Line**
In 2-3 sentences: For your practice setting, which guideline is more applicable and why? Where does genuine uncertainty remain?

### Rapid guideline update summary

Use this when a society issues a focused update (not a full guideline revision) and you need to quickly understand what changed and what it means for your practice.

<PromptPlayground tool="Claude" title="Rapid Guideline Update Summary">
You are a clinical guidelines expert summarizing a focused guideline update for a physician audience.

**Purpose:** Distill a focused update into what changed, why, and what to do about it. Skip everything that stayed the same.

### Rules
- Focus exclusively on new or changed recommendations.
- Preserve original recommendation grades.
- Note the specific new evidence that prompted the update.
- Do not fabricate recommendations or evidence grades.

### Output format

**Update Overview**
- Organization, year, scope of update (what triggered it, what it covers).

**What Changed**

| # | Prior Recommendation (Grade) | New Recommendation (Grade) | Key New Evidence | Clinical Impact |
|---|------------------------------|----------------------------|------------------|-----------------|
| 1 | [Old rec] | [New rec] | [Study/data that prompted change] | [What to do differently] |

**What Did NOT Change** (brief)
One paragraph confirming that remaining recommendations are unchanged.

**Clinical Bottom Line**
In 1-2 sentences: The one thing you need to know from this update.

---
**FOCUSED UPDATE TEXT:**
[Paste the focused update text here]

**PRIOR FULL GUIDELINE TEXT (if available):**
[Paste relevant sections of the prior full guideline for comparison]
</PromptPlayground>

### Specialty-specific guideline extraction

Use this when a guideline covers a broad clinical domain and you need to extract only the recommendations relevant to your specific specialty or practice setting.

<PromptPlayground tool="Claude" title="Specialty-Specific Guideline Extraction">
You are a clinical guidelines expert extracting specialty-relevant recommendations from a broad guideline.

**Purpose:** Filter a large guideline down to the recommendations that matter for a specific practice setting, with practical implementation notes.

### Rules
- Include only recommendations directly relevant to the specified specialty.
- Preserve original recommendation grades.
- Add practical implementation notes for each recommendation in the context of the specified setting.
- Do not fabricate recommendations or evidence grades.

### Output format

**Filtered Recommendations for [SPECIALTY]**

| # | Recommendation | Grade | Why It Matters for [SPECIALTY] | What to Do |
|---|---------------|-------|-------------------------------|------------|
| 1 | [Rec] | [Grade] | [Relevance] | [Action] |

**What You Can Skip**
Brief summary of guideline sections not relevant to this specialty.

**Red Flags / When to Refer**
Recommendations that indicate the patient needs a different specialist or a higher level of care.

**Clinical Bottom Line**
In 1-2 sentences: What does this guideline mean for a [SPECIALTY] clinician?

## Example output

Here is what the main prompt produces when given a realistic guideline summary:

**Input provided:**
The 2023 AHA/ACC/HFSA Guideline for the Management of Heart Failure (focused update).

**LLM output:**

```
╔══════════════════════════════════════╗
║  GUIDELINE OVERVIEW                  ║
╚══════════════════════════════════════╝
```

- **Issuing Body:** AHA/ACC/HFSA
- **Year:** 2023 (focused update of 2022 guideline)
- **Scope:** Management of heart failure, with emphasis on HFrEF pharmacotherapy updates
- **Target Population:** Adults with heart failure (all stages)
- **Methodology:** Systematic evidence review with GRADE-informed recommendations using ACC/AHA Class/Level of Evidence system
- **Funding:** AHA/ACC (no industry funding for guideline development)
- **Conflicts of Interest:** Disclosed per ACC/AHA policy; several panel members reported consulting fees from SGLT2i and ARNI manufacturers

```
╔══════════════════════════════════════╗
║  KEY RECOMMENDATIONS TABLE           ║
╚══════════════════════════════════════╝
```

| # | Recommendation | Class | Level | Clinical Scenario | New/Changed? |
|---|---------------|-------|-------|-------------------|-------------|
| 1 | Initiate GDMT with four pillars (ARNI/ACEi/ARB + beta-blocker + MRA + SGLT2i) in all HFrEF (LVEF ≤40%) | I | A | All HFrEF without contraindications | Updated |
| 2 | SGLT2 inhibitors (dapagliflozin or empagliflozin) recommended for HFrEF regardless of diabetes status | I | A | HFrEF, with or without T2DM | New |
| 3 | SGLT2 inhibitors recommended for HFpEF (LVEF >40%) to reduce HF hospitalizations | IIa | B-R | HFpEF with recurrent volume overload or HF hospitalizations | New |
| 4 | ARNI preferred over ACEi/ARB in most HFrEF patients | I | A | HFrEF tolerating ACEi/ARB | Upgraded |

```
╔══════════════════════════════════════╗
║  WHAT CHANGED FROM PRIOR VERSION     ║
╚══════════════════════════════════════╝
```

1. **SGLT2 inhibitors added as fourth pillar of GDMT** (previously not included). Driven by DAPA-HF and EMPEROR-Reduced trials showing mortality and hospitalization benefits independent of diabetes.
2. **HFpEF now has a pharmacologic recommendation** (SGLT2i, Class IIa). Previously, HFpEF management was limited to diuretics and comorbidity management. Driven by EMPEROR-Preserved and DELIVER trials.
3. **Simultaneous initiation of all four pillars emphasized** over sequential titration. Rationale: early benefit from each class, no evidence that sequential approach is safer.
4. **ARNI upgraded to preferred over ACEi/ARB** (previously positioned as a step-up after ACEi). Driven by growing real-world data supporting first-line use.

```
╔══════════════════════════════════════╗
║  CONTROVERSIES / MINORITY OPINIONS   ║
╚══════════════════════════════════════╝
```

- Simultaneous initiation of four drug classes remains debated. Some panelists noted the evidence for simultaneous vs. sequential initiation comes from registry data, not RCTs.
- The Class IIa recommendation for SGLT2i in HFpEF is stronger than some panelists felt the evidence supported, given that EMPEROR-Preserved and DELIVER showed reduced hospitalizations but no mortality benefit.

```
╔══════════════════════════════════════╗
║  PRACTICE IMPLICATIONS               ║
╚══════════════════════════════════════╝
```

- Every HFrEF patient should be assessed for all four drug classes at every visit. The goal is not "add one drug at a time" but "get to four drugs as quickly as tolerated."
- SGLT2 inhibitors are now indicated in heart failure even without diabetes. Insurance coverage for this indication is still catching up; prior authorization may be required.
- HFpEF patients now have an evidence-based pharmacotherapy option. Start dapagliflozin or empagliflozin in HFpEF patients with recurrent hospitalizations.
- If LVEF recovers above 40% on GDMT, continue all medications. Do not de-escalate based on improved EF.

```
╔══════════════════════════════════════╗
║  CLINICAL BOTTOM LINE                ║
╚══════════════════════════════════════╝
```

The most important change: SGLT2 inhibitors are now a core pillar of heart failure therapy for both HFrEF and HFpEF, independent of diabetes status. Start all four GDMT classes early and simultaneously in HFrEF. For HFpEF, SGLT2 inhibitors are the first drug class with a guideline recommendation.

## Tips & pitfalls

- Guidelines are periodically updated. Always verify you are referencing the most current version before acting on recommendations.
- Some organizations (ACC/AHA, IDSA, ATS, USPSTF) use different grading systems. The AI will attempt to preserve the original system, but verify against the source document.
- This template works best when you paste the full recommendation sections, not just the abstract or executive summary. The more text you provide, the more accurate the extraction.
- The model may assign evidence grades that are not in the original text, or confuse ACC/AHA Class (strength of recommendation) with Level of Evidence (quality of supporting data). Always cross-check grades against the source.
- For focused updates, the model needs the original guideline text to accurately describe what changed. Provide both documents when possible.
- Guideline panels sometimes make strong recommendations based on weak evidence (driven by clinical judgment or ethical considerations). The model should flag these, but verify independently.
- Different societies sometimes issue conflicting recommendations on the same topic. This template summarizes one guideline at a time. For cross-guideline comparison, use the **Conflicting Guideline Comparison** variation or the Evidence Comparison template.
