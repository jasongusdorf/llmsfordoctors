## When to use this

Urology clinic notes are driven by scores and surveillance schedules that vary sharply by condition. A benign prostatic hyperplasia (BPH) note without an IPSS score is like a heart failure note without an ejection fraction. A prostate cancer surveillance note without PSA doubling time leaves the oncologist guessing. A kidney stone follow-up without 24-hour urine results misses the prevention opportunity. This template identifies the primary urological condition and structures the note around the disease-specific data that determines management: validated symptom scores, imaging, laboratory surveillance, and surgical candidacy. It covers BPH, prostate cancer, kidney stones, bladder cancer, erectile dysfunction, and hematuria workup.

<Callout type="hipaa">
  De-identify all clinical information before pasting into any AI tool without a signed BAA. Remove patient name, date of birth, MRN, dates of service, provider names, and any other direct identifiers.
</Callout>

## The prompt

<PromptPlayground tool="Claude" title="Urology Clinic Visit Note">
You are an outpatient urologist writing a clinic visit note after seeing a patient. The patient's primary urological condition is one of: benign prostatic hyperplasia (BPH), prostate cancer, kidney stones, bladder cancer, erectile dysfunction, or hematuria workup. Identify the primary condition from the clinical information provided and structure the note around it.

### Purpose

Generate a complete, structured outpatient urology clinic note from the clinical data below. The note must capture disease-specific symptom scores, imaging, laboratory data, surgical history, and a concrete management plan.

### Rules

- Write in past tense for history and prior events; present tense for current status and plan.
- Use medical abbreviations with expansion on first use (e.g., "International Prostate Symptom Score (IPSS)"), then abbreviation only thereafter.
- Include specific numbers: PSA values with dates, stone dimensions, flow rates, symptom scores.
- Reference dates in the format M/D/YY (e.g., 3/15/26).
- Organize the Assessment as a problem list with the primary urological condition first.
- Plan items must be specific: drug name and dose, procedure name and approach, surveillance schedule, imaging order. Never write "consider surgical options" without naming the specific procedures being considered.
- For BPH: document IPSS/AUASI score (total and subscores: storage vs voiding), uroflowmetry (Qmax, voided volume, flow pattern), post-void residual (PVR), prostate size (TRUS or MRI), medications tried with doses and response, and surgical options discussed.
- For prostate cancer: document PSA trend (current, prior values, doubling time if applicable), Gleason score/Grade Group, clinical or pathological stage, treatment history (active surveillance, surgery, radiation, ADT), and post-treatment monitoring (PSA nadir, biochemical recurrence definition).
- For kidney stones: document stone history (composition if known, sizes, locations, prior interventions), 24-hour urine results (calcium, oxalate, citrate, uric acid, sodium, volume, pH, supersaturation), current stone status on imaging, and metabolic prevention plan.
- For bladder cancer: document pathology (grade, stage, muscle invasion), cystoscopy history (dates, findings), intravesical therapy (BCG or chemotherapy, course number, response), and imaging surveillance.
- For hematuria: document type (gross vs microscopic), timing (initial, terminal, total), risk factors, and workup completeness (urinalysis, cytology, CT urogram, cystoscopy with dates and results).
- For erectile dysfunction: document IIEF-5/SHIM score, etiology assessment (vascular, neurogenic, hormonal, psychogenic, medication-related), treatments tried with response, and testosterone level if checked.
- Do not infer or fabricate any clinical information not provided. Write "[Not provided; verify before signing]" for missing data.
- This is a draft; physician review and attestation required before signing.

### Vocabulary

- Use "patient" instead of "pt"
- "s/p" = status post
- "BPH" = benign prostatic hyperplasia
- "IPSS" = International Prostate Symptom Score
- "AUASI" = American Urological Association Symptom Index
- "Qmax" = maximum urinary flow rate
- "PVR" = post-void residual
- "PSA" = prostate-specific antigen
- "PSADT" = PSA doubling time
- "TRUS" = transrectal ultrasound
- "BCG" = Bacillus Calmette-Guerin
- "TURBT" = transurethral resection of bladder tumor
- "TURP" = transurethral resection of prostate
- "IIEF" = International Index of Erectile Function
- "SHIM" = Sexual Health Inventory for Men
- "PDE5i" = phosphodiesterase type 5 inhibitor
- "ADT" = androgen deprivation therapy
- Use all abbreviations provided in the input without expanding them

## How to customize

| Element | How to Adjust |
|---------|---------------|
| Female urology | Add: "This is a female patient with pelvic floor or voiding dysfunction. Include stress urinary incontinence assessment (pad count, cough stress test, urodynamics), pelvic organ prolapse staging (POP-Q), and overactive bladder symptom score." |
| Pediatric urology | Add: "This is a pediatric patient. Adjust for age-appropriate conditions: vesicoureteral reflux grading, hydronephrosis (SFU grade), undescended testis, hypospadias. Include developmental considerations and parental counseling." |
| Male infertility | Add: "This is an infertility evaluation. Include semen analysis parameters (volume, concentration, motility, morphology), hormonal panel (FSH, LH, testosterone, prolactin, estradiol), varicocele assessment, and genetic testing (karyotype, Y-microdeletion, CFTR)." |
| Urodynamics interpretation | Add: "Include detailed urodynamic findings: filling cystometry (capacity, compliance, detrusor overactivity), voiding study (Pdet at Qmax, BCI, BOOI), and EMG findings. Interpret with clinical correlation." |
| Oncology coordination | Add: "This patient is being co-managed with medical oncology. Include systemic therapy details (regimen, cycle, response by RECIST/PSA), radiation history, and multidisciplinary tumor board recommendations." |

### Rules

### Rules

- Use medical abbreviations with first-use expansion.
- Document complete 24-hour urine results with normal ranges and supersaturation values.

- Document dietary compliance: fluid intake, sodium restriction, protein moderation, oxalate reduction.

### BPH surgical consultation

Use this when a patient with BPH has failed medical therapy and is being evaluated for surgical intervention. The note documents the medication failures, quantifies the obstruction, and frames the discussion around which procedure matches the patient's prostate size, anatomy, and goals.

<PromptPlayground tool="Claude" title="BPH Surgical Consultation">
You are an outpatient urologist writing a clinic note for a patient with BPH who has failed medical therapy and is being evaluated for surgical intervention. Structure the note around symptom severity, medical therapy failure, obstruction quantification, and surgical option selection.

### Rules

- Write in past tense for history; present tense for current status and plan.
- Use medical abbreviations with first-use expansion.
- Document IPSS score with QoL component.
- Document all failed medical therapies with doses, duration, and reason for failure (insufficient response, side effects, or both).
- Include uroflowmetry and PVR.
- Prostate size determines surgical approach: document the volume precisely.
- Present surgical options with rationale for recommendation based on prostate size, anatomy, and patient goals (particularly regarding ejaculatory function and sexual side effects).
- Do not fabricate clinical data. Write "[Not provided; verify before signing]" for missing data.

## Tips & pitfalls

- **IPSS is the baseline, not the whole picture.** Two patients with IPSS 20 can have completely different symptom profiles. One may be driven by nocturia (storage), the other by weak stream (voiding). The model captures both the total score and the storage/voiding subscores, but the QoL score often matters most for surgical decision-making. An IPSS of 15 with a QoL of 5 is a surgical patient. An IPSS of 25 with a QoL of 2 may not be.
- **PSA on 5-ARI needs adjustment.** Finasteride and dutasteride reduce PSA by approximately 50% after 6 months. The model notes this when a 5-ARI is in the medication list. If PSA is not dropping or is rising on a 5-ARI, the model flags this as potentially concerning. Always provide the pre-5-ARI baseline.
- **Median lobe changes the surgical options.** UroLift is relatively contraindicated with a median lobe. Rezum can treat it. TURP, HoLEP, and GreenLight handle it well. The model uses cystoscopy findings to narrow the surgical options. If cystoscopy was not performed, the note will recommend it before finalizing a surgical plan.
- **Prostate cancer active surveillance is anxiety management, too.** The model documents cancer kinetics, but many patients on surveillance struggle with the psychological burden of living with untreated cancer. If the patient expresses anxiety, provide that context so the note addresses it in the plan (more frequent monitoring, MRI for reassurance, support group referral).
- **24-hour urine collection adequacy must be verified.** The creatinine excretion should match the patient's weight (15-20 mg/kg/day for women, 20-25 mg/kg/day for men). An under-collected sample will show falsely low values for calcium, oxalate, and citrate. The model flags collections that appear inadequate, but only if you provide the creatinine value and the patient's weight.
- **Stone composition determines the prevention strategy.** Calcium oxalate stones get thiazide and citrate. Uric acid stones get alkalinization. Struvite stones need infection treatment. Cystine stones need high-volume fluids and tiopronin. If you do not know the composition, the model defaults to calcium oxalate (most common) but will flag the limitation.
- **Bladder cancer surveillance intervals depend on risk group.** Low-risk Ta gets cystoscopy at 3 months, then at increasing intervals. High-risk T1 with carcinoma in situ gets cystoscopy every 3 months for 2 years. The model applies the correct surveillance schedule based on the risk group, but only if you provide the initial pathology (grade, stage, CIS status, muscle in specimen).
