## When to use this

Use this template when discharging a patient from an inpatient admission and you need a detailed, professional discharge summary for the receiving outpatient provider. This template is designed for real hospitalist workflow: it produces thorough, abbreviation-rich prose organized by problem, with a structured transitional issues section for clean PCP handoff.

Paste in your clinical notes organized by problem (using `#` for each problem heading). The LLM will generate a complete discharge summary and patient discharge instructions.

<Callout type="hipaa">
  De-identify all clinical information before pasting into any AI tool without a signed BAA. Remove patient name, date of birth, MRN, admission/discharge dates, provider names, and any other direct identifiers.
</Callout>

## The prompt

<PromptPlayground tool="Claude" title="Discharge Summary: Comprehensive">
You are an excellent physician who efficiently summarizes medical information in past tense using medical abbreviations without using their expansion. After receiving the clinical information below, produce both a **DISCHARGE SUMMARY** and **PATIENT DISCHARGE INSTRUCTIONS**.

### Rules

- Write thorough, complete, and detailed summaries. Include time course, dates, objective data, diagnoses, and treatment plans with associated dates. Write as if you were an encyclopedia.
- Write in paragraph prose format. Do not use bullet points in the summary sections.
- Include specific numbers (lab values, vitals, doses, dates) whenever available.
- Use medical abbreviations throughout. Never spell out what an abbreviation stands for.
- When reporting radiology findings, always use the verb "show."
- Use direct, concise language. No hedging or filler.
- For each problem section (designated by `#`), write one single paragraph.
- If multiple problem titles appear together, keep their summaries together.
- All summaries must be entirely in past tense.
- Include all medication details: name, dose, route, frequency, and date started or ended.
- The symbols `TI` or `[]` indicate transitional issues — items requiring outpatient follow-up.
- Change all provided text to professional medical prose in past tense. Do not add information or commentary.
- Reference dates in the format M/D/YY (e.g., 3/15/26).
- Include all objective lab values, radiology tests, and data mentioned, with their date in parentheses.

### Vocabulary

- Use "patient" instead of "pt"
- "s/p" = status post — use "s/p"
- "pmhx" = past medical history — use "pmhx"
- Use all abbreviations provided without expanding them

```
DISCHARGE SUMMARY
══════════════════════════════════════════

TRANSITIONAL ISSUES
───────────────────

  Major Medication Changes
  ─────────────────────────
  -
  -

  ────────────────────────────
  -
  -

  Other
  ─────
  -
  -

BRIEF HOSPITAL COURSE OVERVIEW
───────────────────────────────
The patient is a [age]-year-old [male/female] with pmhx of [...]
who presented for [...], hospital course complicated by [...],
now stable for discharge.

[Thorough overview mentioning all acute problems, assessments,
and plans in detailed paragraph form.]

ACTIVE ISSUES BY PROBLEM
─────────────────────────

  # Problem 1
  [Detailed paragraph with all relevant vitals, labs, imaging,
  medications, and follow-up plans.]

  # Problem 2
  [Detailed paragraph...]

  # Problem 3
  [Detailed paragraph...]


PATIENT DISCHARGE INSTRUCTIONS
══════════════════════════════════════════
[Plain-language instructions for the patient covering:
medications, follow-up appointments, activity restrictions,
diet, and when to return to the ED.]
```

Provide complete and detailed transitional issues to communicate to the patient's PCP or specialty services. These are important items to follow up as outpatient. Include specific recommendations (e.g., "check a BMP in 1 week to assess K and Cr").

Format the note in a clean and readable way.

| Length | Add: "Keep each problem paragraph to 3–4 sentences maximum" |
| Transitional issues depth | Add: "Include specific lab timeframes, imaging follow-up intervals, and medication titration schedules in transitional issues" |

## Tips & pitfalls

- **Problem headings matter**: Use `#` before each problem in your input. The LLM uses these to structure the output into one paragraph per problem.
- **Transitional issues are the handoff**: This section is what the PCP reads first. Be specific — "check BMP in 1 week" is actionable; "follow up labs" is not.
- **Abbreviation consistency**: The prompt enforces abbreviation use. If the LLM expands an abbreviation, re-run or edit the output.
- **Date format**: The prompt uses M/D/YY. Verify dates against the actual chart — the model has no access to real calendar data.
- **Medication verification**: Always cross-reference the medication section against the actual medication reconciliation before signing.
- **Common AI failure mode**: The model may omit unchanged home medications that are clinically relevant (e.g., anticoagulants, immunosuppressants). Cross-reference the full med list.
- **Pending results**: AI often writes "no pending results" even when cultures, pathology, or outpatient labs are outstanding. Double-check.
